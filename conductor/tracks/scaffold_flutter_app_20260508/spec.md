# Track Specification: Scaffold Flutter desktop application and set up basic UI layout

## Goal
Initialize the Flutter project configured for Windows, macOS, and Linux desktop environments, and establish the foundational UI layout referencing the Stitch design.

## Scope
- Create the Flutter app with desktop support enabled.
- Clean up default counter app.
- Implement the basic application shell (window frame, main navigation, central content area).
- Integrate fundamental design tokens (colors, typography) from Stitch reference.

## Out of Scope
- Full implementation of all screens.
- Backend API integration.
